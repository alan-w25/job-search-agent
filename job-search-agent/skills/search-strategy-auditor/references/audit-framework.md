# Audit Framework Reference

Diagnostic frameworks, benchmarks, pattern detection trees, and intervention strategies
for the search-strategy-auditor skill. Load this reference when performing pipeline
analysis, rejection pattern detection, or burnout assessment.

---

## Stage-by-Stage Diagnostic Framework

When conversion drops below benchmark at a specific stage, use this framework to
diagnose probable cause and recommend targeted fixes.

### Stage 1: Application to Screen (Benchmark: 10-20%)

**Below 5% — CRITICAL**

| Probable Cause | Diagnostic Signal | Recommended Fix |
|---|---|---|
| Resume not passing ATS | Applying through portals with no response | Run resume-tailor with ATS optimization for each application. Check for formatting issues (tables, headers, graphics that ATS can't parse). |
| Targeting misalignment | Match scores averaging below 55% | Narrow targeting to roles with 65%+ match scores. You're wasting applications on long shots. |
| Resume lacks impact quantification | Generic bullet points without metrics | Rewrite top 8 bullets with quantified results. "Managed team" becomes "Led 6-person team that shipped $2M product in 4 months." |
| Applying too late | Roles posted 30+ days before application | Set up alerts. Apply within 72 hours of posting. After 2 weeks, your resume enters a much larger pile. |
| Missing keywords | JD analysis shows required skills not on resume | Map JD keywords to resume content. If you have the skill, it must appear in the exact phrasing the JD uses. |

**5-10% — BELOW BENCHMARK**

| Probable Cause | Diagnostic Signal | Recommended Fix |
|---|---|---|
| No cover letter or weak cover letter | Applying to roles that request or benefit from cover letters without one | Use cover-letter-writer for roles where the posting requests one or where you have a specific angle. |
| Weak LinkedIn presence | Applications where hiring team likely checks LinkedIn | Align LinkedIn headline and summary with target role keywords. |
| No referral or networking | 100% cold applications | Aim for 30%+ of applications to include some warm contact (alumni, mutual connection, event follow-up). |
| Resume formatting issues | Getting screens from direct referrals but not from portal applications | Test resume in multiple ATS systems. Simplify formatting. |

**10-20% — HEALTHY**

No action needed at this stage. Monitor for decline.

**Above 20% — STRONG**

Excellent top-of-funnel performance. Ensure you're not under-applying (converting well but
low volume means few total screens).

---

### Stage 2: Screen to Interview (Benchmark: 40-60%)

**Below 20% — CRITICAL**

| Probable Cause | Diagnostic Signal | Recommended Fix |
|---|---|---|
| Poor phone screen performance | Advancing to screens but not interviews | Run interview-coach in Recruiter Coach mode. Practice concise pitch, salary expectation handling, and "tell me about yourself." |
| Salary misalignment | Screen conversations ending after comp discussion | Research market rates before every screen. Have a prepared range, not a single number. Use "I'm flexible depending on total comp" framing. |
| Overqualified signal | Screener questions focused on "why this level?" or "why this company?" | Prepare a compelling "why down/lateral" narrative. If you can't articulate it convincingly, you may actually be misaligned. |
| Underqualified reality | Screens reveal requirements not apparent in JD | This is a targeting issue. Increase weight on must-have requirements in job-analyzer scoring. |

**20-40% — BELOW BENCHMARK**

| Probable Cause | Diagnostic Signal | Recommended Fix |
|---|---|---|
| Lack of enthusiasm | Generic responses, no company-specific research | Run interview-coach Company Deep Dive before every screen. Reference one specific, recent company development. |
| Poor "why this role" answer | Fumbling the motivation question | Write and rehearse a 30-second "why this role at this company" for each screen. |
| Red flags surfacing | Questions about gaps, job hopping, or career changes met with poor responses | Prepare and rehearse responses to your top 3 resume red flags. Every candidate has them. |

**40-60% — HEALTHY**

No action needed. Monitor.

**Above 60% — STRONG**

Screen performance is excellent. If overall conversion is still low, the problem is
upstream (not enough applications) or downstream (interview performance).

---

### Stage 3: Interview to Offer (Benchmark: 20-40%)

**Below 10% — CRITICAL**

| Probable Cause | Diagnostic Signal | Recommended Fix |
|---|---|---|
| Technical skill gap | Failing technical rounds specifically | Identify which technical areas. Invest 2-4 weeks in targeted prep before more applications. Use interview-coach technical prep mode. |
| Behavioral interview weakness | Passing technicals but failing behavioral/culture rounds | Build STAR story bank with 8-10 stories. Practice with interview-coach mock interviews until delivery is natural. |
| System design gap (senior roles) | Failing system design rounds specifically | Study system design fundamentals. Practice 3 full system design problems with interview-coach. |
| Culture misfit pattern | Consistent "not a culture fit" feedback | Re-evaluate company targeting. If enterprise companies keep rejecting you, try startups (or vice versa). |
| Negotiation failures | Getting to offer stage but offers being pulled | This is rare — offers pulled usually means unreasonable counter or red flag behavior. Get coaching before next negotiation. |

**10-20% — BELOW BENCHMARK**

| Probable Cause | Diagnostic Signal | Recommended Fix |
|---|---|---|
| Inconsistent performance | Some strong interviews, some weak | Identify what differs. Energy level? Preparation time? Company type? Find the pattern and standardize your strong approach. |
| Missing "wow factor" | Advancing but never the top candidate | Add a portfolio piece, case study, or "I already thought about your problem" element to interviews. Differentiate. |
| Poor question-asking | Not asking thoughtful questions in return | Prepare 5 research-backed questions per interview. Generic questions signal generic interest. |

**20-40% — HEALTHY**

No action needed. At this range, some rejection is expected and normal — not every role
is the right fit even when you perform well.

**Above 40% — EXCEPTIONAL**

You interview very well. If you're not getting enough interviews, the constraint is
upstream. Focus all effort on pipeline volume and screen performance.

## Industry Benchmarks by Role Type

Conversion rates vary significantly by industry and role type. Use these adjusted
benchmarks when the standard ranges don't fit.

### Technology (Software Engineering, Product, Design, Data)

| Transition | Typical Range | Notes |
|---|---|---|
| Application to Screen | 8-18% | Lower end for FAANG/top-tier (high volume of applicants). Higher end for startups and mid-market. |
| Screen to Interview | 35-55% | Technical screens filter more aggressively than other industries. |
| Interview to Offer | 15-35% | Multi-round interview loops (4-6 rounds) reduce conversion. |
| Overall | 1-6% | High application volume makes this inherently low. 3%+ is good. |

### Finance (Banking, PE, Hedge Funds, Fintech)

| Transition | Typical Range | Notes |
|---|---|---|
| Application to Screen | 5-12% | Extremely competitive. Networking and referrals matter more here than anywhere. |
| Screen to Interview | 45-65% | Once you get a screen, conversion is higher — they pre-filter aggressively. |
| Interview to Offer | 25-45% | Fewer interview rounds than tech. Fit matters enormously. |
| Overall | 1-4% | Referrals can increase overall rate to 8-12%. |

### Healthcare (Clinical, Health Tech, Pharma, Biotech)

| Transition | Typical Range | Notes |
|---|---|---|
| Application to Screen | 12-25% | Specialized skills mean smaller applicant pools. |
| Screen to Interview | 50-70% | Credential-driven screening is more binary. |
| Interview to Offer | 30-50% | Fewer candidates per role means higher conversion. |
| Overall | 3-10% | Specialized roles can see 10%+ overall conversion. |

### Consulting (Management Consulting, Advisory)

| Transition | Typical Range | Notes |
|---|---|---|
| Application to Screen | 5-10% | Prestige-driven, extremely competitive. School and firm pedigree matter. |
| Screen to Interview | 40-55% | Case-based screening is binary: you can do it or you can't. |
| Interview to Offer | 20-35% | Multi-round case interviews with high attrition. |
| Overall | 1-3% | Networking and referrals can 3-4x this rate. |

### General / Cross-Industry (Operations, Marketing, HR, Admin)

| Transition | Typical Range | Notes |
|---|---|---|
| Application to Screen | 10-22% | Broader applicant pools but also broader requirements. |
| Screen to Interview | 45-60% | Standard behavioral screening. |
| Interview to Offer | 25-40% | Fewer rounds, faster decisions. |
| Overall | 2-8% | Most aligned with the standard benchmarks. |

## Velocity Benchmarks by Career Level

Application pace expectations vary by seniority. A senior executive applying to 10 roles
per week is spraying; an entry-level candidate at that pace may be appropriately active.

### Entry Level (0-2 years experience)

| Metric | Benchmark | Notes |
|---|---|---|
| Applications/week | 5-10 | Higher volume acceptable because roles are more interchangeable. |
| Time to tailor per application | 30-60 minutes | Use resume-tailor but don't over-polish. Speed matters at this level. |
| Expected search duration | 2-4 months | Longer if first job, shorter if internship-to-full-time pipeline. |
| Stalling threshold | <3/week | Below this, something is blocking you (perfectionism, unclear targeting). |

### Mid-Level (3-7 years experience)

| Metric | Benchmark | Notes |
|---|---|---|
| Applications/week | 3-7 | Sweet spot. Enough tailoring to be competitive, enough volume for pipeline. |
| Time to tailor per application | 45-90 minutes | More customization expected. Hiring managers at this level read resumes carefully. |
| Expected search duration | 1-3 months | Varies heavily by market conditions and specialization. |
| Stalling threshold | <2/week | Below this, velocity is a constraint. |

### Senior Level (8-15 years experience)

| Metric | Benchmark | Notes |
|---|---|---|
| Applications/week | 2-5 | Quality over quantity. Each application should be highly targeted. |
| Time to tailor per application | 1-2 hours | Significant customization. At this level, generic applications are obvious. |
| Expected search duration | 2-5 months | Senior roles have longer hiring cycles. Don't panic at month 3. |
| Stalling threshold | <1/week | Even 1 well-targeted application per week keeps pipeline alive. |

### Executive Level (VP+, C-Suite, 15+ years)

| Metric | Benchmark | Notes |
|---|---|---|
| Applications/week | 1-3 | Most exec roles are filled through networking, not applications. |
| Time to tailor per application | 2-4 hours | Each application is essentially a pitch deck for your candidacy. |
| Expected search duration | 3-9 months | Normal. Executive searches move slowly on both sides. |
| Stalling threshold | <1 per 2 weeks | At exec level, networking activity matters more than application count. |

## Pattern Detection Decision Tree

Use this tree to diagnose rejection patterns across multiple applications. Start at the
root and follow the branches based on data.

```
ROOT: Map all rejections to gap types (Skill / Signal / Fit / Process)
|
+-- Is one gap type >60% of rejections?
|   |
|   +-- YES: Dominant pattern detected
|   |   |
|   |   +-- Dominant = SKILL
|   |   |   |
|   |   |   +-- Is the gap concentrated in 1-2 specific skills?
|   |   |   |   +-- YES: "The market is telling you {{SKILL}} is table stakes.
|   |   |   |   |    Close this gap or reposition away from roles requiring it."
|   |   |   |   +-- NO: "Broad skill gaps suggest seniority miscalibration.
|   |   |   |        Consider targeting one level lower or investing in
|   |   |   |        skill development before continuing."
|   |   |   |
|   |   +-- Dominant = SIGNAL
|   |   |   |
|   |   |   +-- Are rejections concentrated at a specific stage?
|   |   |   |   +-- Resume stage: "Your resume undersells your experience.
|   |   |   |   |    Run resume-tailor with impact-focused rewrites."
|   |   |   |   +-- Interview stage: "You have the skills but aren't
|   |   |   |   |    communicating them under pressure. Run interview-coach
|   |   |   |   |    mock sessions — this is a practice problem, not a
|   |   |   |   |    qualification problem."
|   |   |   |   +-- Mixed stages: "Positioning overhaul needed. Your entire
|   |   |   |        narrative — resume, LinkedIn, interview delivery — needs
|   |   |   |        alignment around your strongest value proposition."
|   |   |   |
|   |   +-- Dominant = FIT
|   |   |   |
|   |   |   +-- Is misfit concentrated by company type?
|   |   |   |   +-- YES: "You keep targeting {{COMPANY_TYPE}} but your profile
|   |   |   |   |    signals {{DIFFERENT_TYPE}}. Either reposition or change
|   |   |   |   |    targeting."
|   |   |   |   +-- NO: "Scattered fit issues suggest you may be presenting
|   |   |   |        differently in each application. Develop a consistent
|   |   |   |        professional identity."
|   |   |   |
|   |   +-- Dominant = PROCESS
|   |       |
|   |       +-- "Your rejections are primarily logistical — slow responses,
|   |            missed follow-ups, scheduling issues. Fix the mechanics:
|   |            set up tracking, templates, and calendar blocks for
|   |            application management."
|   |
|   +-- NO: Mixed pattern (no type >60%)
|       |
|       +-- Are rejections concentrated at one stage?
|           +-- YES: "You consistently drop off at {{STAGE}}. This is a
|           |    {{STAGE}}-specific issue regardless of gap type. Focus all
|           |    improvement effort on {{STAGE}} performance."
|           +-- NO: "Your rejections are distributed across types and stages.
|                This usually means targeting is the root issue — you're
|                applying too broadly. Narrow your focus to roles where you
|                have 70%+ match scores."
```

## Burnout Risk Indicators and Intervention Strategies

### Risk Level Assessment

Score each indicator as present (1) or absent (0):

| # | Indicator | How to Detect |
|---|---|---|
| 1 | Velocity decay | Applications/week declining for 3+ consecutive weeks |
| 2 | Rejection accumulation | 3+ consecutive rejections with no strategy adjustment |
| 3 | Engagement gaps | 10+ day gaps between applications with no stated reason |
| 4 | Quality decay | Match scores trending downward (applying to worse-fit roles) |
| 5 | Follow-up abandonment | Active applications not being followed up on |
| 6 | Perfectionism paralysis | Spending 3+ hours per application without submitting |
| 7 | Avoidance behavior | Starting applications but not finishing them |
| 8 | Desperation signals | Sudden shift to applying for roles significantly below level |

### Risk Levels

| Score | Level | Response |
|---|---|---|
| 0-1 | LOW | No intervention needed. Normal search friction. |
| 2-3 | MODERATE | Acknowledge pattern. Suggest process adjustments (see below). |
| 4-5 | HIGH | Direct conversation about search sustainability. Recommend restructuring approach. |
| 6+ | CRITICAL | Recommend pausing active applications for 1 week. Focus on strategy reset, not volume. |

### Intervention Strategies by Risk Level

**MODERATE (2-3 indicators):**
- Reduce weekly target by 30-50%. 3 excellent applications beats 7 mediocre ones.
- Set fixed "search hours" — e.g., 9-11am weekdays. Outside those hours, the search
  does not exist.
- Review your top 3 strongest active applications. Invest follow-up energy there instead
  of new applications.
- Celebrate process milestones, not just outcomes. "Submitted 3 tailored applications
  this week" is a win regardless of callbacks.

**HIGH (4-5 indicators):**
- Everything in MODERATE, plus:
- Revisit career profile. Has your target shifted? Are you still excited about the roles
  you're targeting? If not, that's the real problem.
- Analyze your best-performing applications (furthest in pipeline). What made them
  different? Double down on that pattern.
- Consider whether you need a skill-building pause. If the market keeps saying you need
  {{SKILL}}, 4 weeks of focused learning may be higher-ROI than 4 more weeks of
  applications.
- Talk to someone in your target role. An informational interview can re-energize and
  provide intel that improves targeting.

**CRITICAL (6+ indicators):**
- Everything above, plus:
- Pause new applications for 5-7 days. This is not giving up — it's preventing a spiral
  where bad applications lead to more rejections which lead to worse applications.
- During the pause: run a full targeting audit, update career profile, refresh resume
  with resume-tailor, and research 5 companies you'd genuinely be excited to join.
- When restarting, begin with exactly 1 application to the strongest-fit role you
  identified during the pause. Nail it. Let that momentum build.
- If total applications exceed 50 with sub-2% conversion, consider whether a fundamental
  change is needed: different role type, different industry, additional credential, or
  professional career coaching.

## Sample Audit Report Structure

Below is an annotated example of what a complete audit report looks like. Use this as a
template when generating reports.

```markdown
# Search Strategy Audit — [DATE]

## Executive Summary

Your search shows strong top-of-funnel performance (18% application-to-screen rate) but a
significant leak at the interview stage (12% interview-to-offer rate, benchmark is 20-40%).
Combined with a declining application velocity (down from 5/week to 2/week over the past
month), your pipeline is drying up. The primary issue is interview performance, not
targeting or resume quality.

## Pipeline Health

| Stage | Your Rate | Benchmark | Status |
|---|---|---|---|
| Application to Screen | 18% | 10-20% | PASS |
| Screen to Interview | 52% | 40-60% | PASS |
| Interview to Offer | 12% | 20-40% | FAIL |
| Overall | 1.1% | 2-8% | WARNING |

**Drop-off diagnosis:** Interview-to-offer conversion at 12% is the primary leak. Of 8
interviews that did not convert, 5 cited "stronger candidates in behavioral rounds" and 2
cited "technical depth concerns." This maps to a signal gap (behavioral) and skill gap
(technical depth) at the interview stage.

## Targeting Audit

**Targeting Health Score: 72/100**

| Dimension | Finding | Impact |
|---|---|---|
| Average match score | 68% | PASS (above 60% threshold) |
| Role type distribution | 2 related types (backend eng, platform eng) | PASS |
| Company stage alignment | 70% growth-stage, matches profile | PASS |
| Seniority calibration | 25% of applications 1 level above profile | WARNING (-10) |
| Geographic alignment | 3 on-site applications conflict with remote preference | WARNING (-10) |
| Match score std deviation | 22 | WARNING (-10) — inconsistent targeting |

## Velocity Analysis

**Classification: DECLINING — was HEALTHY, trending toward STALLING**

| Week | Applications | Trend |
|---|---|---|
| Week 1 | 5 | -- |
| Week 2 | 6 | UP |
| Week 3 | 4 | DOWN |
| Week 4 | 2 | DOWN |

Follow-up status: 2 applications overdue for follow-up (submitted 14+ days ago, no
response, have hiring manager contact).

## Rejection Patterns

**Dominant pattern: Signal gap at interview stage (62% of rejections)**

Breakdown:
- Signal gap (interview): 62% — you have the skills but aren't demonstrating them
  under interview pressure
- Skill gap (system design): 25% — consistent feedback about system design depth
- Fit gap: 13% — minor, not actionable at this sample size

Root cause: behavioral interview delivery. Your STAR stories lack specificity and
quantified impact based on available feedback.

## Market Reality Check

**Alignment: ALIGNED**

- Target salary range ($180-220K for senior backend roles) is within market range for
  your geography and experience level (market median: $195K, P75: $225K).
- Backend engineering demand is stable in your target market (posting volume flat
  month-over-month).
- No major hiring freezes or layoffs in your target company set.

## Strategy Recommendations

[PRIORITY: 1] [POSITIONING]
WHAT: Build a STAR story bank with 10 stories, each with quantified impact, practiced
to natural delivery.
WHY: 62% of your rejections are signal gaps at the interview stage. You have the skills
but are not communicating them.
EXPECTED IMPACT: Improving interview-to-offer rate from 12% to 25% would yield 1
additional offer per 8 interviews.
EFFORT: Medium (4-6 hours over 1 week)
SKILL TO USE: interview-coach (Mock Interview mode)

[PRIORITY: 2] [SKILL DEVELOPMENT]
WHAT: Complete 3 system design practice problems focused on distributed systems.
WHY: 25% of rejections cite system design depth. This is a recurring, specific gap.
EXPECTED IMPACT: Removing this gap could convert 1-2 of your next 8 interviews.
EFFORT: High (8-12 hours over 2 weeks)
SKILL TO USE: interview-coach (Technical Prep mode)

[PRIORITY: 3] [PROCESS]
WHAT: Follow up on 2 overdue applications this week. Send a brief, value-adding
check-in (reference something relevant to the role, not just "checking in").
WHY: These are warm leads going cold. Following up costs 10 minutes and can reactivate
stalled processes.
EXPECTED IMPACT: 20-30% of follow-ups reactivate a stalled process.
EFFORT: Low (20 minutes total)
SKILL TO USE: N/A — just send the emails

[PRIORITY: 4] [TARGETING]
WHAT: Remove on-site roles from your search unless they are exceptional fits (90%+
match score). Apply that time to better-matched remote roles.
WHY: 3 applications to on-site roles conflict with your remote preference and are
unlikely to convert even if you advance.
EXPECTED IMPACT: Redirects ~30 minutes/week to higher-probability applications.
EFFORT: Low (adjust search filters)
SKILL TO USE: job-analyzer (use to validate match before applying)

[PRIORITY: 5] [MINDSET]
WHAT: Increase application velocity back to 4/week. Set fixed search blocks:
Tuesday and Thursday 9-11am for applications, Monday and Wednesday 9-10am for
follow-ups and prep.
WHY: Velocity has dropped 60% in 4 weeks. Pipeline will empty if this continues.
EXPECTED IMPACT: Maintaining 4 quality applications/week sustains pipeline for
continued interview opportunities.
EFFORT: Low (calendar commitment)
SKILL TO USE: N/A — discipline and scheduling

## Burnout Check

**Risk Level: MODERATE (3 of 8 indicators present)**

Indicators detected:
- [PRESENT] Velocity decay: 3 consecutive weeks of declining applications
- [PRESENT] Rejection accumulation: 4 recent rejections without strategy change
- [PRESENT] Quality decay: last 3 applications had lower match scores than average
- [ABSENT] Engagement gaps
- [ABSENT] Follow-up abandonment
- [ABSENT] Perfectionism paralysis
- [ABSENT] Avoidance behavior
- [ABSENT] Desperation signals

This is early-stage search fatigue, not full burnout. The recommendations above
address the root causes. If velocity continues to decline next week, consider a
structured 3-day pause to reset.

---

## Top 3 Changes to Make This Week

1. **Follow up on 2 stalled applications** (20 minutes). You have warm leads going cold.
   Send a brief, relevant check-in referencing something specific to each role. Expected
   result: 1 of 2 may reactivate.

2. **Write and practice 5 STAR stories with quantified impact** (3 hours). Your biggest
   pipeline leak is interview-stage signal gaps. This single change addresses 62% of
   your rejections. Use interview-coach mock interview mode to pressure-test delivery.

3. **Submit 3 applications to remote roles with 70%+ match scores** (3-4 hours). Your
   velocity needs to recover and your targeting needs tightening. This combines both
   fixes. Use job-analyzer to validate fit before applying and resume-tailor for each
   submission.
```
